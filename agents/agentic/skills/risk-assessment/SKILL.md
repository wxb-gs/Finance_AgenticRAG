# 风险评估

当前查询涉及风险评估，遵循：

## 工作流程

1. **风险分类**：先将风险归类（财务/经营/合规/市场）
2. **证据收集**：不仅找正面证据，必须主动搜索负面信号
3. **对立分析**：对每个风险点，同时搜索支持和反对的证据
4. **量化评分**：给每类风险打分（1-5），附带依据

## 输出规范

- 每类风险单独评分并说明理由
- 标注信息来源和时效性
- 如发现重大风险信号，显式用 **加粗** 标注
